# Agent Mode Prompts

This directory contains compact, structured prompts optimized for the MCP Agent Mode.

## Structure
- `orchestrator.md`: Main system prompt for the agent.
- `planner.md`: Prompt for generating execution plans.
- `coder.md`: Prompt for code generation.
- `reviewer.md`: Prompt for code review and validation.

## Guidelines
- Use JSON output format where possible.
- Keep context minimal (use lockfiles).
- Focus on single-responsibility tasks.
